# Financial Risk Analysis: EDA Case Study (Group 13)
This project applies Exploratory Data Analysis (EDA) to a retail banking scenario to identify patterns that indicate whether a loan applicant is likely to default. By analyzing historical data, we aim to help consumer finance companies minimize financial loss while ensuring that capable applicants are not unfairly rejected.

The analysis focuses on identifying key risk factors such as education level, occupation, and gender that correlate with payment difficulties.
